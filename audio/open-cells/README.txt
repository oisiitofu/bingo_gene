Put custom OPEN sounds here.

Supported filenames:
- 1.mp3, 2.mp3, 3.mp3 ...
- 01.mp3, 002.wav, 0024.wav, etc.
- 1_1.mp3, 1_2.wav, 24_1.mp3, etc.

Supported extensions: mp3, wav, ogg, m4a.
If multiple files match the opened cell number, one is picked at random.
Files must start with the cell number to be matched.
If you add files while the page is open, the app checks the folder again the next time that number is opened.
When opening index.html directly in Chrome/Edge, the app cannot read this folder list, so it tries likely filenames directly.
